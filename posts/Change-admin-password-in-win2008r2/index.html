Windows Server 2008 R2에서 관리자 비밀번호 변경 이후 처리

Change admin password in Windows Server 2008 R2

주기적으로 서버의 admin password를 변경해야 할 일이 있을 때 적용 가능한 스크립트 입니다.

이 스크립트가 변경 가능한 항목은 다음과 같습니다.

  • 서비스의 시작 계정
  • Application pool 의 프로세스 시작 계정
  • Web site 의 디렉토리 접근 계정
  • Web application 의 디렉토리 접근 계정
  • Virtual directory 의 디렉토리 접근 계정
  • *.config 파일에 저장된 비밀번호

config 파일은 따로 목록을 정리하여 csv 파일로 스크립트에 제공합니다. config 비밀번호 변경 시 비밀번호에 $가들어가는경우에는가 들어가는 경우에는가들어가는경우에는로 치환이 됩니다. 따라서, $$ 를 넣고 싶은 경우에는 $$$ 로 넣어 주어야 합니다.

config 파일 목록은 다음과 같은 형식으로 저장됩니다.

> Get-Content .\configList.csv
hostname,filePath
HOSTNAME,C:\webroot\web.config

스크립트가 변경하지 못하는 항목은 다음과 같습니다.

  • configList에 없는 config 파일의 비밀번호
  • 작업 스케줄러 일부(GUI로 작성한 일부 작업 스케줄러의 경우 [매개 변수가 틀립니다] 오류가 발생)
  • 원격 서버 설정 변경

스크립트를 실행하고 나면 config 파일을 제외하고 적용된 항목을 csv 파일로 저장합니다.

저장되는 항목은 다음과 같습니다.

  1. id: 비밀번호 변경한 id
  2. name: 대상의 이름
  3. hostname
  4. type: WinService, Scheduled Task, Application Pool, Web Site, Web Appplication, Virtual Directory 중 하나
  5. status: Started / Running / Stopped

전체 소스는 다음과 같습니다.

## powershell v2.0 에서는 foreach 대상이 되는 배열이 null이더라도 한번 실행이 되는 현상이 있음...
## foreach 실행 전에 if($arr -ne $NULL) 이런식으로 null 체크를 해야한다

## string 저장할때 ""(double quota)로 저장하면 $$ 가 변수 이름으로 치환됨
## ''(single quota)로 저장
## 비밀번호 변경 시 escape가 될 가능성을 다시 확인할 필요 있음

# id 정규표현식으로 찾을 때 id가 흔한 패턴인 경우 정규표현식의 상세화를 해 주거나, 앞뒤의 .*를 제거
# id : serviceAdmin pw : Ehdhgodud0!@# 의 경우
$idRegex = '.*(serivceAdmin).*'
$pwBeforeRegex = '이전비밀번호'
$id = 'domain\serivceAdmin'
$pw = '변경할비밀번호'

## 서버 서비스 목록
$services = Get-WMIObject win32_service | where {$_.StartName -Match $idRegex}

## 작업 스케줄러 목록(작업 스케줄러는 운영체제 언어가 한글인 경우 멤버 프로퍼티들이 한글이름으로 나옴)
$schds = schtasks /Query /V /FO CSV | ConvertFrom-Csv | Where {$_."다음 사용자 이름으로 실행" -Match $idRegex -or $_."Run As User" -Match $idRegex }

## Application Pool 목록
## IIS 쪽 작업을 위해서는 아래 모듈을 등록해야함
Import-Module WebAdministration
$pools = Get-ChildItem -Path IIS:\AppPools | Where {$_.processModel.userName -Match $idRegex}


## Web Site 목록
$sites = Get-Website | Where {$_.userName -Match $idRegex}

## Web Applcation 목록 (Collection[n].userName , password)
## Collection[0] = web Application root
$applications = Get-WebApplication | Where {$_.Collection[0].userName -Match $idRegex}

## Virtual Directory 목록
$virtualDirs = Get-WebVirtualDirectory | where {$_.userName -Match $idRegex}

## 위에서 조회한 목록을 엑셀로 추출
$hostname = hostname
$export = @()

if($services -ne $NULL) {
    foreach($item in $services) {
        $element = new-Object PSObject -Property @{
            type = "WinService"
            hostname = $hostname
            name = $item.DisplayName
            id = $item.StartName
            status = $item.State
        }
        $export += $element
    }
}
if($schds -ne $NULL) {
    foreach($item in $schds) {
        $element = new-Object PSObject -Property @{
            type = "Scheduled Task"
            hostname = $hostname
            name = ""
            id = ""
        }
        if( $item."작업 이름" -ne $NULL ) {
            $element.name = $item."작업 이름"
            $element.id = $item."다음 사용자 이름으로 실행"
        }else {
            $element.name = $item.TaskName
            $element.id = $item."Run As User"
        }
        $export += $element
    }
}
if($pools -ne $NULL) {
    foreach($item in $pools) {
        $element = new-Object PSObject -Property @{
            type = "Application Pool"
            hostname = $hostname
            name = $item.name
            id = $item.processModel.userName
            status = $item.State
        }
        $export += $element
    }
}
if($sites -ne $NULL) {
    foreach($item in $sites) {
        $element = new-Object PSObject -Property @{
            type = "Web Site"
            hostname = $hostname
            name = $item.name
            id = $item.userName
            status = $item.State
        }
        $export += $element
    }
}
if($applications -ne $NULL) {
    foreach($item in $applications) {
        $element = new-Object PSObject -Property @{
            type = "Web Application"
            hostname = $hostname
            name = $item.path
            id = $item.Collection[0].userName
        }
        $export += $element
    }
}
if($virtualDirs -ne $NULL) {
    foreach($item in $virtualDirs) {
        $element = new-Object PSObject -Property @{
            type = "Virtual Directory"
            hostname = $hostname
            name = $item.path
            id = $item.userName
        }
        $export += $element
    }
}
$export | Export-Csv $hostname'.csv' -Encoding UTF8
#######################################################

## 비밀번호 재설정

## Config file 비밀번호 변경 - 미리 준비한 list에 있는 비밀번호를 변경
$configList = Import-Csv '.\configList.csv'
$todayString = Get-Date -Format yyyyMMdd
if($configList -ne $NULL) {
    foreach($item in $configList) {
        if($item.hostname -eq $hostname) {
            $configFilePath = $item.filePath
            Copy-Item $configFilePath -Destination $configFilePath"."$todayString".bak"
            $fileContent = Get-Content $configFilePath
            $fileContent -Replace $pwBeforeRegex, $pw | Set-Content -Path $configFilePath -Encoding UTF8
        }
    }
}
## 서비스
## stop에서 4:4:3 은 시스템 메시지 남기는 용도, 숫자의 의미는 크게 없음
## Restart-Service로 대체
if($services -ne $NULL) {
    foreach($sv in $services) {
        sc.exe config $sv.Name obj= $id password= $pw
        if($sv.State -eq "Running") {
            #sc.exe stop $sv.Name 4:4:3
            #sc.exe start $sv.Name
            Restart-Service $sv.Name
        }
    }
}
## 작업 스케줄러 > 한/영 구분하여 처리
if($schds -ne $NULL) {
    foreach($sc in $schds) {
        if($sc."작업 이름" -ne $NULL) {
            schtasks.exe /Change /TN $sc."작업 이름" /RU $id /RP $pw
        }elseif ($sc.TaskName -ne $NULL) {
            schtasks.exe /Change /TN $sc.TaskName /RU $id /RP $pw
        }
    }
}

## Application Pool
if($pools -ne $NULL) {
    foreach($pool in $pools) {
        $pool.processModel.userName = $id
        $pool.processModel.password = $pw
        $pool | Set-Item
    }
}
## from http://stackoverflow.com/questions/25215354/how-to-set-connectas-user-in-iis-using-powershell
## Web Site
if($sites -ne $NULL) {
    foreach($site in $sites) {
        $siteName = $site.Name
        $fullPath = "system.applicationHost/sites/site[@name='$siteName']/application[@path='/']/virtualDirectory[@path='/']"
        Set-WebConfiguration $fullPath -Value @{username=$id;password=$pw}
    }
}
## Web Application
if($applications -ne $NULL) {
    foreach($app in $applications)
    {
        $xpath = $app.ItemXPath
        $fullPath = "$xpath/virtualDirectory[@path='/']"
        Set-WebConfiguration $fullPath -Value @{username=$id;password=$pw}
    }
}
## Virtual Directory
if($virtualDirs -ne $NULL) {
    foreach($vdir in $virtualDirs)
    {
        $xpath = $vdir.ItemXPath
        $fullPath = $xpath.Substring(1)
        Set-WebConfiguration $fullPath -Value @{username=$id;password=$pw}
    }
}

$end = Read-Host "Please enter key for exit"
